Penambangan Data


Mata kuliah Penambangan Data (Data Mining) membahas teknik dan metode untuk mengekstraksi pengetahuan yang bermanfaat dari kumpulan data yang besar. Mahasiswa akan mempelajari konsep dasar, proses penambangan data, serta berbagai metode seperti klasifikasi, klastering, asosiasi, regresi, dan deteksi anomali


Selain teori, mata kuliah ini menekankan pada penerapan praktis menggunakan perangkat lunak dan algoritma untuk menganalisis data dalam berbagai bidang seperti bisnis, kesehatan, dan teknologi. Hasil dari penambangan data diharapkan dapat membantu dalam pengambilan keputusan berbasis data


-Macam Macam Tipe Data-

A. Tipe data berdasarkan pengukurannya

1. Nominal : Data kategori tanpa urutan tertentu, contoh : (biru,hijau,merah)
2. Ordinal : Data kategori dengan urutan tertentu, contoh : (tinggi & rendah)
3. Interval : Data numerik dengan perbedaan bermakna tanpa nol absolut, contoh : (celcius,fahrenheit)
4. Rasio : Data numerik dengan nol absolut, sehingga bisa dibandingkan secara proporsional, contoh : (berat badan, tinggi badan & penghasilan)

B. Tipe data berdasarkan formatnya

1. Numerik : Data angka yang bisa dihitung, contoh : (integer & float)
2. Kategorikal : Data berbentuk kategori atau label, contoh : (pegawai,wirausaha,mahasiswa)

11. Masukkan kode untuk menentukan hasil Outlier

    Kode ini digunakan untuk menentukan outlier pada data Iris menggunakan metode Euclidean Distance dan menyimpan hasilnya ke dalam file Excel


  • pandas untuk membaca dan menyimpan data Excel
  • numpy untuk operasi matematika dan array
  • cdist dari scipy.spatial.distance untuk menghitung jarak Euclidean antar titik data
  • Menambahkan kolom "Is_Outlier" → True jika outlier, False jika bukan outlier


import pandas as pd
import numpy as np
from scipy.spatial.distance import cdist

# Baca data dari Excel 
df = pd.read_excel("/content/drive/My Drive/kombi_iris_final.xlsx")

# Ambil hanya kolom fitur (tanpa ID & Class)
X = df[['petal_length', 'petal_width', 'sepal_length', 'sepal_width']]
print(X.head())  # Cek hasilnya

# Hitung Jarak Euclidean untuk Semua Titik
distances = cdist(X, X, metric='euclidean')

K = 4  # Ubah jumlah tetangga terdekat menjadi 4
sorted_distances = np.sort(distances, axis=1)  # Urutkan jarak
avg_k_nearest = np.mean(sorted_distances[:, 1:K+1], axis=1)  # Ambil rata-rata 4 terdekat

# Hitung Outlier (Gunakan Mean + 2*Std)
threshold = np.mean(avg_k_nearest) + 2 * np.std(avg_k_nearest)
df["Outlier_Score"] = avg_k_nearest
df["Is_Outlier"] = df["Outlier_Score"] > threshold

# Tampilkan Hasil
print(df[["id", "Outlier_Score", "Is_Outlier"]])

# Simpan ke Excel
df.to_excel("hasil_outlierrr.xlsx", index=False)
